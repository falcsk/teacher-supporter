# 小学校教育サポーター

iPadのClaudeアプリでリポジトリを選択し、スラッシュコマンドで授業準備を行うためのプロンプト集です。

## 使い方

1. iPadのClaudeアプリでこのリポジトリを選択
2. 以下のスラッシュコマンドを入力して起動
3. 指導書・教科書の画像を送る
4. 対話しながら板書計画を確定
5. 最後に「プロンプト出して」→ Nano Bananaに1枚ずつ貼り付け

## コマンド一覧

| コマンド | 教科 | 説明 |
|---|---|---|
| `/sansuu` | 算数 | 問題解決型授業の板書計画 |
| `/kokugo` | 国語 | 物語文・説明文・書く活動の板書計画 |
| `/dotoku` | 道徳 | 資料・発問・子どもの意見整理の板書計画 |
| `/ongaku` | 音楽 | YouTube検索・手遊び・活動中心の板書計画 |

## ワークフロー

```
指導書画像を送る
    ↓
対話しながら授業設計（3ステップ）
    ↓
「プロンプト出して」と入力
    ↓
Nano Banana用プロンプトが4枚分出力される
    ↓
Nano Banana（Gemini画像生成）に1枚ずつ貼り付け
    ↓
板書画像完成 → Goodnotesで微調整
```

## 対象学年

デフォルト：小学3年生（コマンド内で変更可）
